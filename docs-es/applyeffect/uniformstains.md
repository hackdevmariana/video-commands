# uniformstains

Crea manchas de color uniforme de la imagen original.

Uso:

``` sh
applyeffect uniformstains imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_uniformstains.png`
