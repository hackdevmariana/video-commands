# noiseperlin

Añade una nebulosa RGB sobre la imagen.

Uso:

``` sh
applyeffect noiseperlin imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_noiseperlin.png`
