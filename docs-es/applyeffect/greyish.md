# greyish

Cambia el tono de la imagen a grisáceo, manteniendo los contornos.

Uso:

``` sh
applyeffect greyish imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_greyish.png`
