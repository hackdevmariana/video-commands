# paleamber

Cambia el tono de las zonas claras de la imagen a ámbar pálido.

Uso:

``` sh
applyeffect paleamber imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_pale_amber.png`
