# texturizepaper

Aplica una textura de papel rugoso sobre la imagen.

Uso:

``` sh
applyeffect texturizepaper imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_texturize_paper.png`
