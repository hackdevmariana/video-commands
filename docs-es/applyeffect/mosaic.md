# mosaic

Unifica las zonas de color de la imagen con un efecto mosaico.

Uso:

``` sh
applyeffect mosaic imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_mosaic.png`
