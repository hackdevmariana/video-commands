# shadowpatch

Aplica una textura de nubes sombreadas sobre la imagen.

Uso:

``` sh
applyeffect shadowpatch imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_shadow_patch.png`
