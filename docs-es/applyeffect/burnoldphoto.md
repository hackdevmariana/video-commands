# burnoldphoto

Crea una textura de foto vieja con los colores quemados.

Uso:

``` sh
applyeffect burnoldphoto imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_burn_old_photo.png`
