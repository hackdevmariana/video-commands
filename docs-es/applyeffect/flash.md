# flash

Expande y aclara las zonas claras de la imagen, quemándola como con flash.

Uso:

``` sh
applyeffect flash imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_flash.png`
