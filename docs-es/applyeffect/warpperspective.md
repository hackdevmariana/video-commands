# warpperspective

Aplica una inclinación en perspectiva de la imagen.

Uso:

``` sh
applyeffect warpperspective imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_warp_perspective.png`
