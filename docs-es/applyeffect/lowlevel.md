# lowlevel

Aumenta el contraste de los colores de la imagen.

Uso:

``` sh
applyeffect lowlevel imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_lowlevel.png`
