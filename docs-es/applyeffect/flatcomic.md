# flatcomic

Unifica y aplana las zonas de color de la imagen.

Uso:

``` sh
applyeffect flatcomic imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_flatcomic.png`
