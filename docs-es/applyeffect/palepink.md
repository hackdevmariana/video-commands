# palepink

Cambia el tono de las zonas claras de la imagen a rosa.

Uso:

``` sh
applyeffect palepink imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_pale_pink.png`
