# subtlewhirls

Crea una textura de ondas sobre la imagen.

Uso:

``` sh
applyeffect subtlewhirls imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_subtlewhirls.png`
