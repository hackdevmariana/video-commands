# arrayfade

Crea una matriz de la imagen original difuminando los bordes.

Uso:

``` sh
applyeffect arrayfade imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_arrayfade.png`
