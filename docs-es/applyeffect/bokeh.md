# bokeh

Dibuja diversos círculos sobre la imagen, generando un efecto Bokeh.

Uso:

``` sh
applyeffect bokeh imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_fx_bokeh.png`
