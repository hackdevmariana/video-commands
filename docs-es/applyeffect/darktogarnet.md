# darktogarnet

Cambia los tonos oscuros de la imagen a granate.

Uso:

``` sh
applyeffect darktogarnet imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_dark_to_garnet.png`
