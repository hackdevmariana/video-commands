# paleblue

Cambia el tono de las zonas claras de la imagen a azul pálido.

Uso:

``` sh
applyeffect paleblue imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_pale_blue.png`
