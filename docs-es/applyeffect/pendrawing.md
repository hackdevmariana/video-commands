# pendrawing

Cambia la textura de la imagen a líneas diagonales.

Uso:

``` sh
applyeffect pendrawing imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_pendrawing.png`
