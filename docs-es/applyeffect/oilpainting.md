# oilpainting

Aplica una textura de pintura al óleo sobre la imagen.

Uso:

``` sh
applyeffect oilpainting imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_oil_painting.png`
