# amberoff

Tiñe de un color ambar apagado las zonas claras de la imagen.

Uso:

``` sh
applyeffect amberoff imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_amber_off.png`
