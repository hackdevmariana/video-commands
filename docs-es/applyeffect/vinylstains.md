# vinylstains

Unifica los colores de la imagen en manchas.

Uso:

``` sh
applyeffect vinylstains imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_vinylstains.png`
