# colorabstraction

Abstrae los colores de la imagen.

Uso:

``` sh
applyeffect colorabstraction imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_color_abstraction.png`
