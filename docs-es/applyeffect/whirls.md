# whirls

Aplica una textura de ondas blanquecinas sobre la imagen.

Uso:

``` sh
applyeffect whirls imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_whirls.png`
