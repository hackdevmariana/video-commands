# waxstains

Aplica una textura a la imagen como manchas de pinturas de cera.

Uso:

``` sh
applyeffect waxstains imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_wax_stains.png`
