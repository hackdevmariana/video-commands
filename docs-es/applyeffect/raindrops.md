# raindrops

Aplica una textura de gotas de agua sobre la imagen.

Uso:

``` sh
applyeffect raindrops imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_raindrops.png`
