# pinetree

Cambia el tono de las zonas claras de la imagen a verde pino.

Uso:

``` sh
applyeffect pinetree imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_pine_tree.png`
