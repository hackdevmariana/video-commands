# waxpaint

Aplica una textura a la imagen como pintado con pinturas de cera.

Uso:

``` sh
applyeffect waxpaint imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_wax_paint.png`
