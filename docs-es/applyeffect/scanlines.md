# scanlines

Crea una textura de líneas oscuras sobre la imagen.

Uso:

``` sh
applyeffect scanlines imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicará el sufijo `_scanlines.png`
