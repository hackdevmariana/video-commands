# reddens

Aplica tonos rojizos a las zonas oscuras de la imagen.

Uso:

``` sh
applyeffect reddens imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_reddens.png`.
