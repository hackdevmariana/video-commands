# intensifies

Reduce la gama, intensificando las zonas oscuras de la imagen.

Uso:

``` sh
applyeffect intensifies imagen_original [imagen_destino]
```

Si no se indica un nombre para el fichero destino, aplicar√° el sufijo `_intensifies.png`
